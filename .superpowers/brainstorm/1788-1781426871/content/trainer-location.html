Where does the trainer live?

Where should reps access the voice call trainer from?

A

Standalone sidebar page — /trainer

A dedicated nav item. Rep picks a lead + persona from a list, then starts a session. Lives alongside Dialer, Tasks, etc.

Pros

  • Full-screen space for hints panel + call UI
  • Easy to discover

Cons

  • Rep has to navigate away from the lead they were looking at
B

Launched from LeadModal — "Practice Call" button

A button inside the lead detail modal opens the trainer in a dialog/drawer with that lead pre-loaded. No new nav item.

Pros

  • Lead context is already there — zero extra clicks
  • Natural pre-call workflow

Cons

  • Cramped inside a modal
  • Harder to discover
C

Both — "Practice Call" in LeadModal opens /trainer?leadId=…

The LeadModal has a "Practice Call" button that deep-links to the standalone trainer page with that lead pre-selected.

Pros

  • Best of both — discoverable + contextual entry point
  • Full screen space for the trainer UI

Cons

  • Slightly more to build
