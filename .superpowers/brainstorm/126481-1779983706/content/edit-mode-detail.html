How it works

  • Click Edit in the header → whole panel switches to edit mode
  • All scripts become editable (title + body text fields)
  • Each category has a "+ Add script" button at the bottom
  • Each script has a 🗑 Delete button
  • Save persists everything to the database for the whole org
  • Cancel discards all unsaved changes
  • The Edit button is hidden for regular Users — only Managers & Admins see it
